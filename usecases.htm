CHARTMOGUL <-> ZOHO CRM 
Integration between CHARTMOGUL AND ZOHO CRM 
Prepared by Mrs. SHANTHINI 
OVERVIEW & PURPOSE 
Using Enrichment API, Add SalesReprestative as Custom Attribute in Chatmogul. 
Create/Update Contact Information while adding the deal in Zoho CRM. 
NEEDED 
1.ZOHO CRM ACCOUNT  
2.CHARTMOGUL ACCOUNT 
Authentication 
1.Zoho CRM - OAuth 2.0 
2.Chart Mogul - Account Token & Secret Key. 
API USED: 
CHARTMOGUL: 
1.Add custom attributes. 
2.Search for customers. 
3.Create customer. 
USE-CASES 
1.It triggers automatically when every New Contact Property Change in Zoho CRM.  
2.You can associate a sales rep with that contact/deal. 
3.Add custom attribute to the customer in chart mogul. 
4.The name of the attribute must be same as in the Zoho CRM. 
1  
5.It is a person name, so the field type is STRING. 
6.Finally, you need to connect the two email fields which need to match between 
Zoho CRM and ChartMogul.
